# 🏠 StayFinder-Airbnb-Clone (MERN Stack)

A full-stack Airbnb Clone built using the **MERN Stack (MongoDB, Express.js, React, Node.js)**. This project allows users to explore, create, and manage property listings with a responsive and dynamic user interface.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 🏡 Add, Edit, Delete Listings (CRUD)
* 🔍 Browse & Search Properties
* 📱 Responsive Design
* ⚡ Fast & Interactive UI
* 🔄 REST API Integration

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB
